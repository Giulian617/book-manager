# Book Manager

## Functionalities

1. **Add Book**  
   The system must allow users to add a new book with a title, ISBN, and publication date.

2. **List Books**  
   The system must display a list of all books stored in the system.

3. **Add Authors**  
   The system must allow users to add new authors with a first name and last name.

4. **Add Categories**  
   The system must allow users to add new book categories.

5. **Assign Publisher**  
   The system must allow users to assign a publisher to a book.

6. **Book Relationships**  
   The system must allow a book to be linked to multiple authors, editors, and translators.

7. **Category Assignment**  
   The system must allow a book to be assigned to one or more categories.

8. **Edit Book Details**  
   The system must allow users to update the details of a book, including:
   - Title  
   - Publisher  
   - Categories  
   - Associated authors, editors, and translators  

9. **Search Books**  
   The system must allow users to search for books by:
   - Author
   - Category

10. **Publisher and Editor Management**  
    The system must allow users to manage publishers and editors independently of books, including:
    - Adding  
    - Editing  
    - Deleting
